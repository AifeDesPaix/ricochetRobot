Sous Linux :
------------

Méthode Graphique :

Faites Clic droit sur "Install_RicochetRobot.sh" puis dans l'onglet "permission", cochez la case "est executable".
Puis Lancez "Install_RicochetRobot.sh" par double clic.
Un fichier va apparaitre : RicochetRobot.sh
LE JEU EST INSTALLÉ.
_______________________________________________________________________________________________

Méthode Console :

Allez dans le répertoire "RicochetRobot" où est présent le fichier "Install_RicochetRobot.sh".
Tapez : $ chmod u+x Install_RicochetRobot.sh
Puis :  $ ./Install_RicochetRobot.sh
LE JEU EST INSTALLÉ.
